v-if VS v-show

V-show doesn't support template element and it doesn't work with v-else. It still remains in the DOM, v-show only toggles the display CSS property.
V-if it ensures that event listeners and child components inside the block are properly destroyed and re-created.
Prefer v-show if you toggle something often and v-if if the condition is unlikely to change at runtime.

Using v-if and v-for together is not recommended
